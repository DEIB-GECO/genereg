Welcome to genexpreg_cancer's documentation!
============================================
**genexpreg_cancer** is a Python library that provides a complete method for analyzing the regulation systems of target genes belonging to genomic pathways relevant for a specific cancer type, assessing the effect on their expression of multiple heterogeneous regulatory elements.
